# Gagarin [![built with gulp](gulp-badge.png)](http://gulpjs.com)

<img width="200" height="200" src="https://raw.githubusercontent.com/neonick/gagarin/master/src/img/logo.png" title="Gagarin frontend template">

## Installation

```
npm install
bower install
gulp
```
